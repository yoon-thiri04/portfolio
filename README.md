# Yoon Thiri Aung — Portfolio

A React + Vite + Tailwind single-page portfolio.

## Run locally

```bash
npm install
npm run dev
```

## Project structure

```
index.html
src/
  main.jsx      # React entry point
  App.jsx       # the portfolio itself
  index.css     # Tailwind entry
vite.config.js
tailwind.config.js
```
